## Initial Setup

```
cargo install tauri-cli
```

## PreREq

```
node --version
v18.14.1
npm --version
9.3.1
```

## Create the app (one time setup)

```
sh <(curl https://create.tauri.app/sh) yaanai-react --template react --manager npm
```

```
cd yaanai-react
npm install
npm run tauri dev
```

## Run

```
npm run tauri dev
```
